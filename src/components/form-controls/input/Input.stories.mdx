## Options
All of the options are avaliable on each of the different types of `<Input />`. (And you may enable all of them at the same time.) For the icon on the left of the `Input`, and in the button on the right, any of the `SystemIcons` are available to you.

An `<Input />` can have help text.
<Canvas>
  <Story name="with Help Text">
    <Input
      helpText="I am helpful text."
      htmlType="text"
      label="Input with Help Text:"
      name="withHelpText"
    />
  </Story>
</Canvas>

An `<Input />` can have helpful icons. Any of the icons available in `<SystemIcon />` are available.
<Canvas>
  <Story name="with an Icon">
    <Input
      htmlType="text"
      icon="edit"
      label="Input with an Icon:"
      name="withIcon"
    />
  </Story>
</Canvas>

An `<Input />` can have an attached button. Any of the icons available in `<SystemIcon />` are available inside the button.
<Canvas>
  <Story name="with a Button">
    <Input
      btnOptions={{
        btnClick: action(`'Input with a Button' button onClick event triggered.`),
        btnIcon: 'navigate',
      }}
      htmlType="text"
      label="Input with an Attached Button:"
      name="test"
    />
  </Story>
</Canvas>

An `<Input />` can have everything at the same time.
<Canvas>
  <Story name="with Everything">
    <Input

</Canvas>
