## Options

A `<Select />` can have help text.
<Canvas>
  <Story name="with Help Text">

         name: 'option 1',
         value: 1,
       },
       {
         name: 'option 2',
         value: 2,
        },
     ]}
    />
  </Story>
</Canvas>

A `<Select />` can have helpful icon. Any of the icons available in `<SystemIcon />` are available.
<Canvas>
  <Story name="with an Icon">
    <Select

         name: 'option 1',
         value: 1,
       },
       {
         name: 'option 2',
         value: 2,
        },
     ]}
    />
  </Story>
</Canvas>

A `<Select />` can have an attached button. Any of the icons available in `<SystemIcon />` are available inside the button.
